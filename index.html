ğŸŒ Project Ecosystem Overview

The JARVIS ecosystem consists of four interconnected core projects, each serving a specific purpose while sharing resources and capabilities:

1

SentinelDB - Intelligent Local Database

Purpose: Custom lightweight database engine optimized for logging, memory persistence, and smart retrieval.

  • Custom .sdb or .ndb format with append-only storage
  • Time-series queryable logs with indexing
  • Query Language: FIND action WHERE context = "deployment"
  • Optional file encryption with JSON and binary modes
  • Integration-ready for Flask/FastAPI
2

Cerebro Shell - Smart CLI + AI Interpreter

Purpose: Natural language terminal assistant that converts intent into shell commands and workflows.

  • Translate user intent into actionable CLI commands
  • Support multiple command sets (Linux, Git, Vercel, Docker)
  • Auto-generate files (vercel.json, .env, .gitignore, etc.)
  • MERN scaffolder with habit detection
  • AI-enhanced explanation capabilities
3

Codexa IDE - Smart Developer Environment

Purpose: Personal developer IDE that adapts to user habits and workflows.

  • Monaco editor with comprehensive language support
  • Side-by-side Git viewer and integrated terminal
  • AI assistant for code explanation and completion
  • SentinelDB integration for preference memory
  • Plugin-ready architecture
4

Jarvis Core - Main AI Assistant

Purpose: Voice-controlled AI operating layer with multi-modal inputs and contextual intelligence.

  • Wake word + voice authentication
  • Whisper-based STT and LLM-based Intent Handler
  • Gesture control and secure WiFi device control
  • Learning Engine with behavioral pattern recognition
  • Modular plugin system

  • Porcupine by Picovoice for offline hotword detection

  • App continuously listens but only activates when:
